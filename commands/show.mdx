---
title: "stdai show"
description: "Display full artifact detail"
---

`stdai show` retrieves and displays a single artifact by its ID. You can view the full human-readable detail, output raw content only, or get the complete artifact as JSON.

## Arguments

<ParamField path="id" type="string" required>
  The artifact ID to display. Prefix matching is supported — you can pass the first few characters of the ID as long as the prefix is unique in the store.
</ParamField>

## Flags

<ParamField path="--json" type="boolean">
  Output the full artifact as a JSON object, including all metadata and signature fields.
</ParamField>

<ParamField path="--content-only" type="boolean">
  Print only the raw artifact content, with no metadata or formatting. Useful for piping the content to another command.
</ParamField>

## Prefix matching

You don't need the full artifact ID. Pass a unique prefix and stdai resolves it:

```bash
# Full ID
stdai show 01HXYZ1234ABCDEF567890ABCD

# Prefix — works if unique
stdai show 01HXYZ
```

If the prefix matches more than one artifact, stdai returns an error.

stdai show 01HXYZ1234ABCDEF567890ABCD

### Raw content only

```bash
stdai show 01HXYZ1234ABCDEF567890ABCD --content-only
```

Prints the content bytes to stdout with no surrounding formatting. Suitable for piping:

```bash
stdai show 01HXYZ1234ABCDEF567890ABCD --content-only | wc -w
```
